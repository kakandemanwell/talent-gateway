# EPRC Jobs Portal

## Getting Started

### 1. Clone the Repository

```bash
git clone <repository-url>
cd eprc-jobs-portal
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the Development Server

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (default Vite port) or `http://localhost:8080`.
